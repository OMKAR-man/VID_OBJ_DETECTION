# 🎥 Video Object Detection

A real-time object detection system for video streams.

## 🚀 Features
- Real-time detection
- Webcam & video support
- Bounding boxes with labels
- High performance

## 🛠️ Tech Stack
- Python
- OpenCV
- YOLO (v5/v8)

## ▶️ How to Run
pip install -r requirements.txt  
python detect_video.py  

## 📌 Applications
- Surveillance
- Traffic monitoring
- Smart systems
